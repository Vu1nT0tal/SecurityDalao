# SecurityDalao

安全大佬 Top 100

> 只是一个突发奇想，包含 Top 100 和历年 Top 10，仅用作膜拜Orz，不具有权威性。
> 
> 数据来自爬虫，每月自动更新，当前版本：2022-04-17
> 
> 欢迎添加各种子榜～

```sh
$ ./install.sh  # 安装

$ ./main.py             # 仅更新数据
$ ./main.py --readme    # 更新数据和README
```
